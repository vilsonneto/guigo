# guigo

Bot WhatsApp para gestão financeira pessoal com painel administrativo.

## Stack
- **Monorepo:** Turborepo (apps/bot, apps/admin, packages/shared)
- **Backend:** Supabase (PostgreSQL), Evolution API (WhatsApp)
- **Frontend:** Next.js (admin panel)
- **Deploy:** Railway (bot) + Vercel (admin)

Em desenvolvimento ativo. Ver [Issues](../../issues) para roadmap.
